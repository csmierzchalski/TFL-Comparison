# Transport & Environmental Data Analysis
RStudio project analysing Transport for London (TfL) journey types, London's monthly average temperatures and congestion charge vehicle volumes. Cleans, visualises and compares real-world datasets to identify trends, seasonal patterns and potential correlations.

## Features

- Data cleaning
- Statistical analysis
- Data visualisation
- Trend identification
- Correlation analysis

## Technologies

- R
- RStudio
- ggplot2
- dplyr
- readr

## Learning Outcomes

- Data analysis
- Statistical methods
- Data visualisation
- Working with real-world datasets
